Surgimento dos computadores

Voltar

Primeira geração

O primeiro computador surgiu em 1944, seu nome era Harvard Mark I. Ele era um computador eletromecânico, ele era uma máquina de cálculo bem grande. Mesmo Mark I sendo o primeiro computador criado, a ideia de criar um computador já existia há muito tempo. Depois de dois anos, em 1946, o computador alemão Colossus chega, também servia para cálculos, mas foi criado especificamente para criptoanálises. No mesmo ano surge o ENIAC nos EUA, um computador automático.

O termo "bug" surgiu nessa época, pois os computadores esquentavam muito, as válvulas também, e com isso atraía insetos, eles pousavam nos equipamentos e quebravam, tinham que consertar manualmente, assim surgiu o termo.

Segunda geração (1959 - 1965)

Nessa época as válvulas evoluíram para transitores, o que fez o tamanho dos computadores diminuírem drasticamente, pra epóca, foi uma grande evolução, pois os computadores se tornaram "pórtateis".

Terceira geração (1965 - 1970)

Na terceira geração surgiram os circuitos integrados, ou seja, os chips, que diminuíram ainda mais o tamanho mas aumentando a eficácia

Quarta geração (1971 - dias atuais)

Os circuitos integrados passaram a ser microprocessadores, os chips ainda são usados, porém continuamos evoluindo, a quarta geração é marcada pelos personal computers, na quarta geração Bill Gates surge com o Altair BASIC, Steve Jobs com a Apple... Isso fez com o que os computadores ficassem cada vez mais populares e mais acessíveis

Surgimento da Internet

Voltar

Na década de 1950, houve a Guerra Fria, constituída entre EUA e URSS. Nessa época as tecnologias explodiram, pois uma potência queria ser mais evoluída do que a outra, o que resultou na corrida espacial, os EUA utilizaram o ENIAC para calcular rota de mísseis, tinham que calcular baseado em muitas variaveis, como temperatura e pressão atmosférica, e o ENIAC era muito bom em fazer isso. Nesse meio surgiu a DARPA (Defense Advanced Research Projects Agency).

Em 1989 Tim Berners-Lee aparece, naquela época as redes pertenciam ao exército, era tudo científico, não havia chance para a população aproveitar da internet. Então, unindo o HTML com TCP/IP, ele criou a World Wide Web (WWW).

Sobre o HTML

Voltar

Primeiramente o HTML, que significa HyperText Markup Language, consiste em três partes.

  1. Tag chamada <html>

  2. Tag chamada <head>

  3. Tag chamada <body>
A estrutura fica assim:

<html>
<head></head>
<body></body>
</html>

Tudo o que estiver entre a Tag <body> é o que está visível para o usuário.

Tudo o que estiver entre a Tag <head> será carregado antes de exibir para o usuário.

Dentro da Tag <head> temos outra Tag chamada <title>, que é basicamente o título da página

Explicando sobre as Tags

Voltar

No HTML existem vários tipos de Tags, cada uma com uma função específica por exemplo: <i>. Essa Tag transforma o texto em itálico.

A tag <strong> deixa o texto em negrito.

Algumas tags não tem fechamento </>. Como por exemplo: <input>

Essa tag cria um campo para digitar.

Tag <h1>: Cria o título.

Tag <h2>: Cria o subtítulo. Até a tag <h6> vão ser todos subtítulos, porém, um menor do que o outro.

Tag <p>: Cria um parágrafo.

Tag <u>: Tudo o que estiver dentro dessa tag, vai ser sublinhado.

Tag <ol>: Cria uma lista ordenada.

Tag <li>: Precisa estar dentro da tag <ol> ou <ul> para funcionar.

Tag <ul>: Cria uma lista normal, sem ordem.

Tag <blockquote>: Serve para criar citações, essa tag cria um espaço a mais. Fica melhor se usar a tag <i> para ficar em itálico.

Tag <sub>: A tag "Subscript Element" ou subscrito é utilizado para escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo.

Tag <sup>: Este é um elemento inline destinado a marcar fragmentos de texto que devam ser renderizados em alinhamento vertical mais alto em relação ao texto principal.

Tag <hr>: Tag sem fechamento, usada para criar um linha ate o final da página para separar alguma coisa.

Tag <small>: Cria um texto menor do que os demais.

Tag <font>: É utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página HTML.

Tag <abbr>: Ela faz abreviações de palavras.

Tag <mark>: Ela grifa palavras.

Tag <del>: Representa uma parte do texto que foi excluída de um documento. Este elemento é (não necessariamente) renderizado pelos navegadores com uma linha entre o texto.
